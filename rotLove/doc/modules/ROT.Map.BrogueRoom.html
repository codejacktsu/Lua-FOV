BrogueRoom:isValid (gen, isWallCallback, canBeDugCallback)
Use two callbacks to confirm room validity.

Parameters:

  • gen userdata The map generator calling this function. Lack of bind() function requires this. This is mainly so the map generator can hava a self reference in the two callbacks.
  • isWallCallback function A function with three parameters (gen, x, y) that will return true if x, y represents a wall space in a map.
  • canBeDugCallback function A function with three parameters (gen, x, y) that will return true if x, y represents a map cell that can be made into floorspace.

Returns:

    boolean true if room is valid.
